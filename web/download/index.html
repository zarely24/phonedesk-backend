PhoneDesk

Download the PhoneDesk app

Install it on the computer the phone stays plugged into. You'll enter a pairing code from your manager to bring the phone online.

🪟 Download for Windows 🍏 Download for macOS

Setup · about 2 minutes

  1. Open the app — Windows: extract the zip, then run PhoneDesk (if Windows warns, click More info → Run anyway). Mac: right‑click the app → Open the first time.
  2. Plug the Android phone in by USB with USB debugging turned on, and tap Allow on the phone.
  3. Type your pairing code → the phone goes online. Keep this computer on while it's in use.

Trouble? Ask your manager to re‑send your pairing code.
